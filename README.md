# C implementation of mutalk

[specification](http://github.com/mutalk/spec)

# Build

## Requirements

* C11 compiler
* POSIX system
* CMake 3.7+

## Compile

Download repo and unpack.

Then

```shell
cmake -DCMAKE_BUILD_TYPE=Release .
make
sudo make install # or use package wrapper for your OS
```
